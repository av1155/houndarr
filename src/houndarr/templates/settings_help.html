Missing Search Controls

  • Batch Size: max missing items considered per cycle (default: 2).
  • Sleep (minutes): wait between cycles (default: 30).
  • Hourly Cap: max successful missing searches/hour (default: 4, 0 = unlimited).
  • Cooldown (days): retry wait per item (default: 14).
  • Unreleased Delay (hrs): wait after release date before searching (default: 36).

Cutoff Upgrade Controls

  • Cutoff search: enables cutoff-upgrade pass (default: off).
  • Cutoff Batch: max cutoff items/cycle (default: 1).
  • Cutoff Cooldown: retry wait for cutoff items (default: 21 days).
  • Cutoff Cap: max successful cutoff searches/hour (default: 1, 0 = unlimited).

Cutoff has its own budget and cooldown so it does not compete with missing searches.

Safe Starting Preset

Start conservative: Batch 2, Sleep 30, Hourly Cap 4, Cooldown 14, Unreleased Delay 36, Cutoff Off, Cutoff Batch 1, Cutoff Cooldown 21, Cutoff Cap 1.
